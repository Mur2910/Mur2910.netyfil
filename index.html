Tutoriais de Hospedagem

Como hospedar no Render

1. Crie uma conta

Visite render.com e crie uma conta gratuita com GitHub, GitLab ou Google.

2. Conecte repositório

No dashboard, clique em "New" → "Static Site" e conecte seu repositório.

3. Configure deploy

Defina nome, branch, build command e publish directory.

4. Deploy automático

Clique em "Create Static Site" e seu site estará online em minutos.

Como hospedar no Vercel

1. Login

Visite vercel.com e faça login com GitHub, GitLab ou Bitbucket.

2. Importe projeto

Clique em "Add New" → "Project" e selecione seu repositório.

3. Configuração automática

O Vercel detecta automaticamente a estrutura do seu projeto.

4. Deploy

Clique em "Deploy" e seu site estará online instantaneamente.

Como hospedar no Netlify

1. Crie conta

Visite netlify.com e crie uma conta gratuita.

2. Arraste e solte (simples)

Para sites simples, arraste a pasta do site para a área de deploy.

3. Conecte Git (recomendado)

Clique em "Add new site" → "Import an existing project".

4. Configure e deploy

Defina build commands e publish directory, então clique "Deploy".
